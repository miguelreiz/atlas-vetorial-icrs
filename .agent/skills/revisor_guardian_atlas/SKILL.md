## 2. CHECKLIST COMPLETO POR CAPÍTULO

### CH-000 — Prefácio
- [ ] Posicionamento do Atlas como obra científica original (não apenas clínica)
- [ ] V_End apresentado como hipótese confirmada por FEM (não como especulação)
- [ ] Sistema de 5 vetores descrito com hierarquia causal correta
- [ ] Tom adequado para público: cirurgiões oftalmologistas + pesquisadores

### CH-001 — Anatomia Corneana
**Histologia (Mente 6):**
- [ ] Epitélio = ~50 µm = ~9% da espessura total (não exagerar)
- [ ] Membrana de Bowman = ~10 µm = LINHA (não banda visível)
- [ ] Estroma anterior (~135 µm = ~25%) = isotrópico, entrelaçado ("feltro")
- [ ] Estroma posterior (~330 µm = ~60%) = ortogonal, lamelas paralelas
- [ ] Proporção OBRIGATÓRIA: anterior:posterior = 1:3 (não 1:1, não 1:2)
- [ ] Membrana de Descemet = ~10 µm = LINHA (não banda)
- [ ] Endotélio = ~10 µm = monocamada de células hexagonais
- [ ] WAXS: 66% fibras em ±22,5° no centro (Meek & Boote, 2009) ✅
- [ ] Rigidez anterior 3× posterior (Brillouin, Scarcelli 2014) ✅
- [ ] Zona ICRS (70–80%): estroma posterior, lamelas paralelas
- [ ] Fibras anteriores: entrelaçadas (isotrópicas) — não "paralelas"
- [ ] Fibras posteriores: paralelas/ordenadas — não "entrelaçadas"

**Biomecânica (Mente 1+6):**
- [ ] Profundidade ótima = 70–75% (não simplesmente "mais fundo = melhor")
- [ ] Razão biomecânica da profundidade: equilíbrio arc-shortening/âncora ✅ FEM
- [ ] Fibrilas colágeno: diâmetro 25–35 nm, periodicidade 67 nm (bandas D)
- [ ] Queratócitos (~10% do volume estromal): sentinelas do remodelamento

### CH-002 — Biomecânica dos Anéis
**ICRS geral (Mente 4):**
- [ ] ICRS a 70–80% profundidade — nunca anterior
- [ ] Perfis: triangular ▲ (Ferrara/Keraring), hexagonal (Intacs), arredondado (Cornealring)
- [ ] Ápice triangular → epitélio (▲ para cima) — REGRA ABSOLUTA
- [ ] Comprimento de arco: 120°–340° disponíveis; 160° é padrão monossegmento

**Arc-shortening (Mente 6+4):**
- [ ] Fibras radiais ACIMA do anel percorrem arco mais longo → tensão ↑
- [ ] Vetor Endotelial: fibras tensionadas puxam superfície anterior para BAIXO (−Z) ✅
- [ ] Superfície sobre o anel = PLANA ou REBAIXADA (nunca "tenting") ✅ FEM 2025
- [ ] Lei de Barraquer: adicionar tecido periférico → aplaina centro ✅
- [ ] Lei de Barraquer: remover tecido central → encurva periferia ✅

**FEM (Mente 2+3):**
- [ ] Modelo: Mooney-Rivlin (não HGO nas versões iniciais — incompatível FEBio 4.12.0)
- [ ] IOP = 0,001999 MPa = 15 mmHg (conversão explícita)
- [ ] Malha: ~480 elementos, tipo penta6+hex8
- [ ] Resultados FEM citados com metodologia ("FEBio 4.12, Mooney-Rivlin, IOP=15mmHg")

### CH-003 — Classificação do Ceratocone
- [ ] 5 padrões Plácido: Nipple (3–5mm), Oval (5–6mm), Globoso (>6mm), Pelúcida (inferior), Pós-LASIK
- [ ] Padrão donut epitelial (Reinstein): epitélio FINO no ápice, ESPESSO na periferia ✅
- [ ] Assinatura Plácido: anéis apertados no K-steep = maior curvatura = menor raio
- [ ] Superfície posterior: protrui ANTERIORMENTE (para dentro) — não "para fora" ❌
- [ ] Fibras no KC: reorientadas 60°/120° — não "quebradas" ou "dissolvidas"
- [ ] Índices: Kmax, Ks, Kmax/Ks, BAD-D, KISA%
- [ ] FEM para KC: Von Mises peak na zona inferotemporal ✅
- [ ] Classificação Amsler-Krumeich se utilizada: graus I–IV com critérios

### CH-004 — Vetor Radial (VR)
**Regra Suprema:**
- [ ] VR = CENTRÍFUGO (← ANEL →) — para fora. Qualquer menção contrária = ❌ CRÍTICO
- [ ] Aplainamento central = CONSEQUÊNCIA do VR (não força VR direta)
- [ ] Superfície sobre o anel = PLANA ou REBAIXADA (✅ FEM 2025)
- [ ] Lei de Barraquer aplicada: anel periférico = aplainamento central ✅
- [ ] V_End = VR vertical descendente (−Z) sobre as fibras radiais tensionadas 🆕
- [ ] Magnitude VR proporcional à espessura do anel (Lei do Disco Mecânico → CH-008)

### CH-005 — Vetor Tangencial (VT)
- [ ] VT age nas EXTREMIDADES do anel (não no corpo central)
- [ ] Corpo central do anel = âncora passiva (não gera VT)
- [ ] Incisão cirúrgica alinhada ao K-steep = VT correto (eixo do anel ⊥ K-flat)
- [ ] Incisão errada → SIA conflitante = "Efeito Nida" ⚠️ (descrito com marcador correto)
- [ ] Acoplamento Poisson: K-flat encurva LEVEMENTE quando K-steep aplaina ✅
- [ ] VT não é simétrico nos dois extremos quando anel em K-steep (extremidades em eixos diferentes)

### CH-006 — Vetor de Torque (Vτ)
- [ ] Vτ = bloqueio das fibras oblíquas (verde, #00CC44)
- [ ] Vτ = freio de progressão do KC ✅ (mecanismo de travamento)
- [ ] Vτ simétrico = sem deslocamento do ápice
- [ ] Vτ assimétrico = gera VComa (→ CH-007)
- [ ] Diferença entre Vτ (travamento) e VT (tangencial extremidades)

### CH-007 — VComa
- [ ] VComa = resultado DIRETO de Vτ assimétrico (não vetor independente)
- [ ] VComa representado como deslocamento posicional do ápice (vetor de posição)
- [ ] Superior > Inferior → ápice desce (VComa descendente)
- [ ] Inferior > Superior → ápice sobe (KC apical)
- [ ] Gerenciado pelo posicionamento do arco (cirurgia de planejamento vetorial)
- [ ] ICE relevante para predição do VComa resultado ✅

### CH-008 — Lei do Disco Mecânico (LDM)
- [ ] LDM: efeito VR ∝ espessura do anel (h) — relação linear aproximada ✅
- [ ] 84% do efeito = altura do perfil do anel (height)
- [ ] Anel 200 µm: ΔK ~1,5–2D médio; Anel 300 µm: ΔK ~2,5–3D médio (variável)
- [ ] LDM não é fórmula exata — é proporção biomecânica (marcar como 💡 se quantificado)
- [ ] Interação LDM × comprimento arco (espessura + arco = nomograma)

### CH-009 — VEsférico
- [ ] VEsférico = componente esférico resultante = soma dos 5 vetores no eixo Z
- [ ] Fórmula: VEsférico ≈ VR + VT·cos(θ) + Vτ·f(assimetria) + V_End
- [ ] ΔK-esférico = ΔKmax aplainado (não ΔKcil)
- [ ] VEsférico ≠ VT (distinção obrigatória)

### CH-010 — ICE (Índice de Coerência dos Eixos)
- [ ] ICE = Kmax_plano_refratométrico / Kmax_mapa_topográfico
- [ ] ICE = 1.0: eixos coerentes (resultado ideal previsível)
- [ ] ICE < 0.6: discordância significativa (resultado menos previsível)
- [ ] ICE não prediz magnitude — prediz confiabilidade da predição ✅
- [ ] Distinção ICE vs outros índices topográficos (BAD-D, KI, ISV) — ICE é Antigravity 💡
- [ ] Estudo retrospectivo ICE: marcar como 💡→🔬 (validação em andamento) ⚠️

### CH-011 — Nomogramas Vetoriais
- [ ] Nomograma baseado em: Kmax, Kcil, fenótipo, paquimetria, ICE
- [ ] Ajuste por paquimetria: <450µm → reduzir espessura
- [ ] Ajuste por fenótipo: Nipple → mono 160°; Oval → dois arcos; Globoso → arcos longos
- [ ] Fórmulas com marcador de evidência: 💡 (empírico) ou ✅ (validado)
- [ ] Segurança: espessura mínima pós-implante ≥ 150 µm abaixo do anel ✅
- [ ] Referência Torquetti et al. para fenótipos ✅

### CH-012 — Casos Clínicos
- [ ] Cada caso com: pré-op (Kmax, Kcil, eixo, paquimetria, ICE), anel selecionado, pós-op
- [ ] Decomposição vetorial de cada caso: qual vetor dominou? resultado previsto vs real?
- [ ] Tabela padronizada por caso (usar `decompositor_caso_clinico`)
- [ ] Pelo menos 4 fenótipos KC representados (Nipple, Oval, Globoso, Ultrafino)
- [ ] Pelo menos 1 caso de complicação com revisão vetorial
- [ ] Referência Nassaralla/Almodin/Sandes para casos brasileiros

### CH-013 — Complicações e Manejo
- [ ] Hipocorreção: VEsférico insuficiente → causa + solução vetorial
- [ ] Hipercorreção: VR excessivo → critério + abordagem
- [ ] SIA conflitante: VT mal direcionado → Efeito Nida definido corretamente
- [ ] Extrusão/exposição: causa biomecânica (paquimetria, profundidade) + manejo ✅
- [ ] Migração: VT desequilibrado → reposicionamento
- [ ] Progressão pós-ICRS: Vτ insuficiente → sinais de alerta
- [ ] Contaminação/infecção: protocolo de manejo (não relacionado ao vetor — OK sem marcador)
- [ ] Classificação da complicação: qual vetor gerou? qual vetor resolve?

### CH-014 — Futuro dos Anéis
- [ ] Tendências com marcador correto (⚠️ para especulativas, 🔬 para em desenvolvimento)

### Dependências Conceituais (Verificar Coerência)
```

  → tensão fibras radiais → equilibrio tensão/curvatura
  → ICRS implantado no posterior (70–80%)
      → arc-shortening fibras acima
          → V_End (descendente −Z) ← OPÕE-SE À PIO

      → Vτ (travamento oblíquas) ← FIBRAS OBLÍQUAS

          → se assimétrico: gera VComa
```

## 10. OS 20 MANDAMENTOS DO REVISOR SUPREMO

```
BIOMECÂNICA INVIOLÁVEL:
1. "Tenting" jamais ocorre sobre o anel — V_End descendente ✅ FEM 2025
2. VR é CENTRÍFUGO (← anel →) — seta para o centro = erro crítico imediato
3. PIO é ASCENDENTE (+Z) — qualquer seta para baixo = erro crítico
4. V_End é DESCENDENTE (−Z) — força que se opõe à PIO
5. ICRS no ESTROMA POSTERIOR (70–80%) — nunca no anterior

ANATOMIA OBRIGATÓRIA:
6. Proporção estromal é 1:3 (anterior:posterior) — nunca 1:1
7. Bowman e Descemet são LINHAS (< 2% da espessura) — nunca bandas
8. Ápice triangular aponta para CIMA (▲→epitélio) — nunca para baixo
9. Epitélio: FINO no ápice KC, ESPESSO na periferia (donut Reinstein)
10. Fibras anteriores: entrelaçadas (isotrópicas) ≠ fibras posteriores: paralelas

EVIDÊNCIA RIGOROSA:
11. Toda afirmação biomecânica tem marcador: ✅ 🔬 💡 ⚠️ ❌
12. ✅ sem referência = abaixar para 💡 ou 🔬
13. "Confirmado por FEM" apenas com especificações do modelo
14. Hipóteses especulativas: marcar ⚠️ com nota de cautela explícita

CLÍNICA SEGURA:
15. Paquimetria < 400µm no local → documentar risco → considerar contraindicação
16. Espessura residual < 150µm → contraindicação absoluta
17. VT mal direcionado (SIA Efeito Nida) → identificar e corrigir antes de publicar
18. Toda complicação tem causa vetorial identificada e solução vetorial proposta

QUALIDADE DE PUBLICAÇÃO:
19. Português primeiro — revisar labels, captions e termos em inglês
20. Consistência global — um valor em CH-001 deve ser idêntico em CH-016
```
