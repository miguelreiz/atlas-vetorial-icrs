## 3. Conceito Biomecânico: Arco Longo vs. Arco Curto

### Arco Curto (90°–120°)
- **VR predomina:** Efeito focal, aplainamento localizado
- **VT baixo:** Poucas setas tangenciais, menor tração meridional
- **Indicação:** Casos com cone focal, K-steep bem definido, máximo VR desejado
- **Risco:** Menor estabilização lamelar (Vτ menor)

### Arco Longo (160°–340°)
- **VT predomina:** Efeito tangencial dominante, redistribuição ampla
- **VR moderado:** Diluído ao longo do arco
- **Indicação:** Ectasia difusa, grandes arcos de astigmatismo irregular, estabilização
- **Vantagem:** Maior Vτ → melhor contenção da progressão
- **Risco de assimetria:** Posicionamento crítico das extremidades (VT)

### Regra Geral de Seleção
```
Ceratocone focal + K-steep definido → Arco curto (120°–160°)
Ectasia difusa + sem K-steep claro → Arco longo (210°–340°)
Assimetria grande (Coma alto) → Segmentos assimétricos (espessuras diferentes)
Miopia alta + KC → Ferrara HM (diâmetro 6,0mm) ou Cornealring 310°
```

## 4. Anéis Concêntricos (Duplex/Triplex)

**Conceito:** Dois ou mais anéis em diâmetros diferentes (ex: r=3,2mm + r=4,8mm) para ampliar o efeito biomecânico.

### Regras de Segurança
- Diferença mínima de espessura entre anéis: **30–50 µm**
- Estroma residual entre anéis: **≥ 120 µm**
- Profundidade mínima: não comprometer Descemet (mín 150 µm de estroma residual posterior)

### Indicações
- Ceratocone avançado com espessura suficiente
- Quando um anel único não alcança a redução desejada do K
- Miopia alta + ectasia

## 5. Planejamento Cirúrgico — Algoritmo de Decisão

```
ENTRADA: Topografia + Paquimetria + Refração + Queixas

PASSO 1 — Identificar o Fenótipo Topográfico:
  A. Cone central (simétrico)         → 2 segmentos simétricos
  B. Cone inferior (assimétrico)      → Segmento inferior mais espesso
  C. Cone lateral-inferior            → Segmento único inferior
  D. Ectasia difusa (>50% da córnea)  → Arco longo (210°+)
  E. Ceratocone + miopia alta         → Modelo HM ou Cornealring grande

PASSO 2 — Determinar a Espessura do Anel:
  Kmax 45–50D: 100–150 µm
  Kmax 50–55D: 150–200 µm
  Kmax 55–60D: 200–250 µm
  Kmax >60D:   250–350 µm (+ avaliar viabilidade)

PASSO 3 — Selecionar o Comprimento de Arco:
  Astigmatismo < 3D: arco 120°–150°
  Astigmatismo 3–5D: arco 150°–210°
  Astigmatismo > 5D: arco 210°+ ou assimétrico

PASSO 4 — Verificar Paquimetria no Ponto de Implantação:
  Espessura mínima no túnel: 400 µm total
  Profundidade alvo: 70–80% (ex: 400 µm espessura → túnel a 280–320 µm)
  Estroma residual posterior: ≥ 150 µm (marg. de segurança)

PASSO 5 — Posicionar o Eixo de Incisão:
  Incisão ALINHADA ao K-steep (meridiano do astigmatismo maior)
  Erro no eixo → SIA conflitante (Efeito Nida)
```

## 6. Checklist de Planejamento Pré-Operatório

- [ ] Topografia com pelo menos 2 mapas reprodutíveis
- [ ] Paquimetria no ponto de implantação planejado
- [ ] Mapa de elevação posterior (excluir ectasia severa)
- [ ] Fenótipo topográfico identificado (1 dos 5)
- [ ] Espessura do anel selecionada conforme Kmax
- [ ] Comprimento de arco definido conforme astigmatismo
- [ ] Diâmetro óptico definido (5,0 ou 6,0 mm)
- [ ] Perfil selecionado (triangular vs arredondado)
- [ ] Eixo de incisão marcado = K-steep
- [ ] Consentimento informado com expectativas realistas
- [ ] Estroma residual calculado (≥ 150 µm posterior ao anel)
