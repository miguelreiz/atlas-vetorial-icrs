#### Posicionamento no Atlas
O Atlas Vetorial ICRS se posiciona como **complementar** a esta obra:
- *Ceratocone: guia completo* → diagnóstico, classificação, visão clínica abrangente
- *Atlas Vetorial ICRS* → biomecânica dos anéis, sistema vetorial, planejamento cirúrgico preciso

As obras se **referenciam mutuamente** — a base clínica de Nassaralla/Almodin/Sandes fornece o contexto diagnóstico; o Atlas fornece o motor biomecânico do tratamento cirúrgico.

## 7. Template de Citação ABNT para o Atlas

### Para artigo em periódico:
```
SOBRENOME, Nome; SOBRENOME, Nome. Título do artigo. Nome do Periódico,
Cidade, v. [volume], n. [número], p. [páginas], [mês abreviado]. [ano].
DOI: [se disponível].
```

### Para livro:
```
SOBRENOME, Nome (org./ed.). Título do livro: subtítulo. Edição.
Cidade: Editora, Ano. [número de páginas] p. ISBN [número].
```

### Para comunicação pessoal:
```
SOBRENOME, Nome. [Informação sobre o assunto]. Cidade, [data].
Comunicação pessoal.
```

### Para tese/dissertação:
```
SOBRENOME, Nome. Título da tese. Ano. [número de folhas] f.
Tese (Doutorado em [área]) — [Instituição], Cidade, Ano.
```
