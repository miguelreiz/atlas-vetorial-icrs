# ICRS Ring Database & Concentric Ring Authority

## Propósito

Esta skill é a **base de dados definitiva** de todos os anéis intracorneanos disponíveis no mercado brasileiro e internacional. Ela fornece:

1. **Catálogo técnico completo** de cada fabricante (modelos, arcos, espessuras, perfis)
2. **Perfil vetorial** de cada configuração (VR, VT, Vτ dominantes)
3. **Filosofias de nomograma** de cada fabricante
4. **Sistema de anéis concêntricos** (duplo e triplo)
5. **Análise comparativa** para decisão cirúrgica

**Material:** PMMA
**Perfil:** Triangular (seção transversal em cunha — base posterior, ápice anterior)
**Base fixa:** 600 µm
**Zona óptica:** 5.0 mm (padrão) e 6.0 mm
**Inventor:** Dr. Paulo Ferrara (Belo Horizonte, MG)

#### Ferrara Padrão — Especificações Completas

| Parâmetro | Valores |
|-----------|---------|
| **Arcos** | 90°, 120°, 150°, 160° |
| **Espessuras** | 150, 200, 250, 300, 350 µm |
| **Base** | 600 µm (fixa em todos os modelos) |
| **Perfil** | Triangular — ápice anterior, base posterior |
| **Diâmetro** | 5.0 mm (padrão); 6.0 mm (disponível) |

#### Anatomia Microscópica do Ferrara Padrão

```
Orientação do corte transversal:

        ▲ ápice (voltado para o epitélio)
       / \
      /   \        Altura = espessura selecionada (150-350 µm)
     /     \       Base = 600 µm (constante)
    /       \
   /__________\    Apoio sobre o estroma POSTERIOR (70-75%)
   
   BASE POSTERIOR (paralela ao endotélio)
```

**Efeito prismático:** O perfil triangular age como um **prisma óptico** — desvia raios de luz para fora do eixo visual. Isto reduz glare e halos intrinsecamente pela geometria, sem necessitar de coating ou tratamento de superfície especial.

**Efeito biomecânico:** O ápice estreito empurrando o estroma anteriormente gera o **tenting mais focal** entre todos os perfis, resultando em VR máximo por unidade de espessura. As arestas vivas nas junções ápice-lateral concentram Von Mises (140–170 kPa), o que explica o haze periférico ocasional.

#### Perfil Vetorial Ferrara Padrão por Configuração

```
Ferrara 90° (arco curto):     VR +++  VT +    Vτ 0    → Nipple puro
Ferrara 120°:                 VR +++  VT ++   Vτ +    → Nipple + astigmatismo
Ferrara 150°:                 VR ++   VT +++  Vτ +    → Oval moderado
Ferrara 160°:                 VR ++   VT +++  Vτ +    → Oval / bow-tie
2 segmentos simétricos:       VR +++  VT +++  Vτ 0    → Globoso / circular
2 segm. assimétricos:         VR +++  VT ++   Vτ ++   → Duck / Snowman
```

#### Nomograma Ferrara

O nomograma Ferrara é historicamente **baseado nos padrões dos anéis de Plácido** (como demonstrado na análise do Cap. 8 — LDM). A lógica é:

1. **Leitura do padrão de distorção** dos anéis de Plácido
2. **Enquadramento em um fenótipo** (nipple, oval, globoso, irregular)
3. **Seleção da espessura** baseada no K-max e no equivalente esférico
4. **Seleção do arco** baseada na extensão da zona ectásica
5. **Definição do eixo** de implantação pelo meridiano mais curvo

> **Nota editorial:** A relação entre o nomograma Ferrara e os anéis de Plácido é discutida na hipótese do Cap. 8 — os nomogramas clássicos convergem nos fenótipos simétricos e divergem nos torsionais.

#### Modelos Especiais

| Modelo | Arco | Espessuras | Indicação Principal |
|--------|------|-----------|---------------------|
| **Ferrara HM** | **>300° (≈320°)** | Até 400 µm | **Alta miopia** + ceratocone |

#### Ferrara HM — Análise Vetorial Especial

O HM é um segmento de **arco ultra-longo (>300°)** — quase um anel completo com uma abertura de ~40° para a incisão. Este design cria um perfil vetorial único:

| Vetor | Magnitude | Mecanismo |
|-------|-----------|-----------|
| **VR** | +++ (alto) | Arco >300° distribui aplainamento por toda a circunferência |
| **VT** | ++++ (muito alto) | Arco cobrindo >83% da circunferência = redistribuição massiva |
| **Vτ** | + (baixo) | Quase simétrico — pouco torque |
| **VComa** | +++ (alto) | Centralização do ápice por tensão circunferencial |
| **ΔSE** | ++++ | Forte correção de equivalente esférico (miopia) |

**Por que o HM corrige miopia tão bem:** O arco >300° cria um efeito de Arc-Shortening quase circunferencial. O efeito Barraquer (adição de volume) é distribuído por toda a periferia, gerando um VR difuso mas potente. Combinado com o VT máximo (comprimento de arco = redistribuição tangencial), o resultado é uma forte redução do equivalente esférico sem gerar torque significativo.

**Nomograma Ferrara:** Historicamente baseado nos padrões dos anéis de Plácido (como demonstrado na análise do Cap. 8 — LDM). A seleção do arco e da espessura é feita pelo enquadramento do padrão topográfico nos modelos canônicos do nomograma proprietário.

### 3. AJL OPHTHALMIC (AJL S.A. — Vitoria-Gasteiz, Espanha)

**Material:** PMMA
**Perfil:** Seção variável (diferencial único)

#### Evolução Histórica

A AJL historicamente fabricava anéis com design similar ao Ferrara. A partir do modelo **PRO** e depois **PRO+**, a AJL diferenciou-se fundamentalmente:

| Modelo | Perfil | Base | Espessura |
|--------|--------|------|-----------|
| **AJL Clássico** | Triangular fixo | 600 µm fixa | Fixa por segmento |
| **AJL PRO** | Triangular otimizado | 600 µm fixa | Progressiva |
| **AJL PRO+** | **Variável** | **600–800 µm variável** | **Progressiva** |

#### AJL PRO+ — O Diferencial

O PRO+ possui **largura de base variável** (0.60–0.80 mm) **E** espessura progressiva ao longo do arco. Essa combinação dupla permite:

- **Base mais larga** (0.80 mm) = maior footprint estromal = maior VR local
- **Base mais estreita** (0.60 mm) = menor footprint = menor VR local
- **Espessura progressiva** = Vτ controlado

**Impacto vetorial:** O PRO+ permite gerar gradientes de VR ao longo do arco do segmento — algo que nenhum outro anel no mercado faz com essa resolução. Em cones irregulares (P5 Complexo), onde a topografia não se encaixa em nenhum padrão canônico, essa flexibilidade é decisiva.

#### Arcos Disponíveis AJL

90°, 120°, 150°, 160°, 210°, 355°

#### Espessuras AJL

150, 200, 250, 300, 350 µm (com variação progressiva nos modelos PRO/PRO+)

#### Filosofia Bicalho — Arco Longo para Quase Todos

O Dr. Frederico Bicalho (criador do Cornealring) defende dois princípios diferenciadores:

**1. Perfil Arredondado = Menos Agressivo**

Diferente dos perfis triangulares (Ferrara, Keraring), o Cornealring tem seção transversal com **bordas suavizadas/arredondadas**. Consequências biomecânicas:

- Menor concentração de Von Mises nas arestas → menor risco de haze
- Melhor distribuição de tensão na interface anel-estroma
- Melhor biocompatibilidade histológica documentada
- Design anti-glare: superfície que difunde luz refletida → menos fotopsias

**2. Arco Longo como Estratégia Universal**

Bicalho defende que arcos longos (220°, 300°, 340°) servem para a maioria dos ceratocones. Justificativa vetorial:

| Arco | VR | VT | Vτ | Indicação |
|------|----|----|-----|-----------|
| 90° | +++ focal | + | 0 | Nipple centrado puro |
| 155° | ++ | ++ | + | Astigmatismo moderado |
| 220° | ++ | +++ | + | Maioria dos KC |
| 300° | ++ | ++++ | + | KC + miopia, globoso |
| 340° | ++ | +++++ | 0 | Alta miopia, quase completo |

**Análise editorial:** A filosofia de arco longo simplifica o planejamento (menos variáveis de decisão) e distribui tensão mais homogeneamente. Funciona melhor quando **VT é prioridade** (astigmatismo, miopia). Funciona menos quando **Vτ é prioridade** (cones descentrados tipo Duck/Snowman onde é preciso torque focalizado).

## PARTE III — ANÉIS CONCÊNTRICOS

### Conceito Biomecânico

A córnea ectásica é um sistema anisotrópico, pré-carregado por PIO, com colágeno desorganizado. **Um único anel cria um vetor dominante de tensão.** Anéis concêntricos criam:

- Um **vetor refrativo interno** (anel 5 mm)
- Um **vetor estabilizador externo** (anel 6 mm)
- Um **campo de redistribuição mais homogêneo**

É **engenharia estrutural multicamada** aplicada à córnea.

### Configuração Dupla (Padrão)

| Componente | Anel Interno | Anel Externo |
|-----------|-------------|-------------|
| **Diâmetro** | 5 mm | 6 mm |
| **Profundidade** | ≈350 µm (ideal 355–365 µm) | ≈400 µm (ideal 390–395 µm) |
| **Função** | **Refrativa** | **Regularizadora** |
| **Atua em** | Miopia e Coma | Distribuição global de tensões |
| **Efeito** | Focal | Biomecânico |
| **Vetor dominante** | VR + VComa | VT + estabilização |

### Configuração Tripla (Casos Severos)

| Componente | Anel 3 mm | Anel 5 mm | Anel 6 mm |
|-----------|----------|----------|----------|
| **Função** | Centralizador | Refrativo | Regularizador |
| **Indicação** | KC severo (Kmax > 65 D) | — | — |
| **Aplicação** | Sequencial: 0 → 6 → 12 meses | — | — |

### Regras de Segurança (Obrigatórias)

```
REGRAS DE SEGURANÇA — ANÉIS CONCÊNTRICOS

1. Diferença mínima entre planos de profundidade: 30–50 µm
2. Estroma residual acima de cada túnel: ≥ 120 µm
3. Profundidade planejada: 0.75 × paquimetria local
4. Faixa ideal geral: 350–400 µm
5. Profundidade máxima permitida: 400 µm
6. Regra dos 60%: não ultrapassar 60% de tecido alterado
7. Distância mínima entre incisões (técnica manual): ≥ 20°
```

### Perfil Vetorial Concêntrico

```
ANEL ÚNICO 5mm:     VR +++   VT +    VComa ++  Estabilidade +
ANEL ÚNICO 6mm:     VR +     VT +++  VComa +   Estabilidade +++
CONCÊNTRICO 5+6mm:  VR +++   VT +++  VComa +++ Estabilidade ++++
TRIPLO 3+5+6mm:     VR ++++  VT +++  VComa ++++ Estabilidade +++++
```

### Indicações Estratégicas

| Indicação | Por que concêntrico | Estratégia |
|-----------|-------------------|-----------|
| **KC central, Kmax > 60-65 D** | Anel único insuficiente | 5mm modula ápice, 6mm estabiliza |
| **Miopia residual em córnea fina** | Evitar hipercorreção focal | Interno = dióptrico, externo = contenção |
| **Pós-CXL com irregularidade** | Córnea rígida mas irregular | 6mm = contenção, 5mm = ajuste óptico |
| **KC severo Kmax > 65 D** | Caso extremo | Triplo concêntrico sequencial |

### Vantagens Documentadas

1. Maior previsibilidade
2. Menor indução de irregularidade periférica
3. Melhor controle do coma
4. Redução mais homogênea de Kmax
5. Menor risco de progressão estrutural

### Riscos e Pontos Críticos

1. Túnel muito superficial → extrusão
2. Planos muito próximos → fraqueza estrutural
3. Excesso de espessura interna → hipermetropização
4. Não respeitar estroma residual ≥ 120 µm

## PARTE IV — ANÁLISE VETORIAL COMPARATIVA: ARCO LONGO vs ARCO CURTO

### Quando o Arco Longo Domina

| Cenário | Arco Longo vence porque |
|---------|----------------------|
| P1 Circular + miopia | VT máximo = ΔSE alto |
| Astigmatismo regular | Redistribuição tangencial ampla |
| KC central + alta miopia | Keraring 325° ou Ferrara HM |
| Córnea fina (risco extrusão) | Tensão distribuída = menos Von Mises focal |

### Quando o Arco Curto Domina

| Cenário | Arco Curto vence porque |
|---------|----------------------|
| P2 Oval descentrado | Precisa de Vτ focalizado, não distribuído |
| P3 Duck | Torque concentrado na zona do cone |
| Nipple puro com coma baixo | VR focal > VR difuso |
| KC assimétrico extremo | Arco longo cobre córnea saudável desnecessariamente |

### Regra Geral do Atlas

```
SE prioridade = VR focal       → arco curto (90-120°) + espessura alta
SE prioridade = VT max          → arco longo (≥210°) + espessura moderada
SE prioridade = Vτ focalizado   → segmento assimétrico (AS) ou arco curto posicionado
SE prioridade = VR + VT + VComa → Ferrara HM (>300°) ou Cornealring 300-340°
SE caso severo (Kmax > 65D)     → anéis concêntricos
```
