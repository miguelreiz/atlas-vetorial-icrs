# 📘 Atlas Vetorial ICRS — Anéis Intracorneanos

> *O primeiro atlas que explica os anéis intracorneanos ao nível das fibras de colágeno.*

---

## 🎯 O Que É Este Atlas

Um atlas editorial e visual que redefine a abordagem aos anéis intracorneanos (ICRS) no ceratocone, substituindo o paradigma antigo ("K-max → tabela → anel") pelo **paradigma vetorial**: cada anel gera **vetores biomecânicos** que agem diretamente nas **fibras de colágeno estromal** — e o disco de Plácido é a **impressão digital** dessas forças.

## 🔬 O Diferencial: Framework Multi-Escala

Cada conceito é apresentado em **3 escalas simultâneas** — algo inédito na literatura de ICRS:

| Escala | O que mostra |
|--------|-------------|
| **MACRO** | Disco de Plácido — o que o cirurgião vê no consultório |
| **MESO** | Corte transversal — o anel implantado no estroma |
| **MICRO** | Fibras de colágeno — a verdade biomecânica por trás de tudo |
