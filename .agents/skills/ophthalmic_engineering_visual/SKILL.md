# Ophthalmic Engineering — Corneal Anatomy & Visual Standards

> **Role:** This skill enables the agent to act as an **ophthalmic medical engineer** — reviewing, correcting, and generating anatomically faithful corneal illustrations for the Atlas. Every image of corneal cross-sections, fiber maps, or pathological progressions must pass through this skill's rules before publication.

---

## 1. Corneal Layers — Accurate Proportions and Properties

All cross-section illustrations MUST respect these proportions and properties:

### Layer Stack (Anterior → Posterior)

| Layer | Thickness | % of Total | Visual Representation | Color Code |
|-------|-----------|------------|----------------------|------------|
| **Epitélio** | ~50 µm | ~9% | Thin smooth line, 5-6 cell layers | Light pink |
| **Bowman** | ~8-12 µm | ~2% | Hairline acellular border | Dark tan line |
| **Estroma Anterior** | ~120-150 µm | ~25% | Dense interwoven mesh, oblique fibers | Warm beige |
| **Estroma Posterior** | ~300-330 µm | ~60% | Parallel ordered lamellae | Light beige |
| **Descemet** | ~10-12 µm (adult) | ~2% | Thin basement membrane line | Gray line |
| **Endotélio** | ~5 µm | ~1% | Single cell monolayer | Light blue |
| **TOTAL** | ~500-550 µm central | 100% | — | — |

### CRITICAL RULES for Cross-Section Illustrations

1. **Posterior stroma MUST be ~2.5-3× thicker than anterior** (not equal)
2. **Epithelium is thin but visible** — never exaggerate to >10% of total
3. **Bowman's layer is a LINE, not a band** — it's only 8-12 µm
4. **Descemet is also a LINE** — similar thickness to Bowman's
5. **Peripheral thickness is ~700 µm** (vs ~550 central) — show this taper
6. **Implantation zone for ICRS** is at 70-75% depth → within the posterior stroma

## 2. Collagen Fiber Directions — Anatomical Reference Map

### Top-Down View (En Face)

```
FIBER ORIENTATION MAP — Normal Human Cornea

                    SUPERIOR (S)
                        |
                        |  ← S-I preferential axis
                        |
    NASAL (N) ──────────●────────── TEMPORAL (T)
                        |       ← N-T preferential axis
                        |
                        |
                    INFERIOR (I)

Centro (zona óptica, ~7 mm):
  66% das fibrilas em ±22.5° dos eixos S-I e N-T
  34% em orientações oblíquas
  γ (grau de orientação) = 0.49 ± 0.10

Transição (~1-2 mm do limbo):
  Fibras ortogonais mudam de direção
  Integram-se com o anel circunferencial limbar

Limbo:
  Fibras predominantemente TANGENCIAIS (circunferenciais)
  Pseudoânulo de colágeno circunda toda a córnea
  Reforço nasal-temporal na periferia
```

### Cross-Section View (Sagittal)

```
FIBER ARCHITECTURE POR PROFUNDIDADE

┌──────────────────────────────────────┐
│  EPITÉLIO (~50 µm)                   │  Sem fibras de colágeno
├──────────────────────────────────────┤
│  BOWMAN (~10 µm)                     │  Fibras aleatórias tipo I, densas
├──────────────────────────────────────┤
│  ESTROMA ANTERIOR (1/3)              │  ISOTRÓPICO
│  ~120-150 µm                         │  Lamelas finas, ramificadas
│  ╲ ╱ ╲ ╱ ╲ ╱  (entrelaçamento)      │  Fibras oblíquas conectam a Bowman
│  ╱ ╲ ╱ ╲ ╱ ╲  (cruzamento 3D)       │  Major interweaving + branching
│                                      │  ALTA RIGIDEZ (3× posterior)
├──────────────────────────────────────┤
│  ESTROMA POSTERIOR (2/3)             │  ORTOGONAL (S-I + N-T)
│  ~300-330 µm                         │  Lamelas largas, full-width
│  ═══════════════  (paralelas)        │  Pouco entrelaçamento
│  ═══════════════  (ordenadas)        │  As lamelas correm de limbo a limbo
│  ═══════════════  (ortogonais)       │  BASSA RIGIDEZ (~39% da anterior)
│                                      │  Fibras elásticas pré-Descemet
├──────────────────────────────────────┤
│  DESCEMET (~10 µm)                   │  Colágeno tipo IV, hexagonal
├──────────────────────────────────────┤
│  ENDOTÉLIO (~5 µm)                   │  Monocamada celular hexagonal
└──────────────────────────────────────┘
```

### RULES for Fiber Direction Illustrations

1. **Anterior fibers:** Draw as criss-crossing, interwoven mesh (like woven fabric). Include oblique/transverse fibers connecting to Bowman's.
2. **Posterior fibers:** Draw as parallel, ordered lines. Adjacent lamellae at ~90° to each other but each runs full width.
3. **NEVER draw anterior and posterior with the same pattern** — they are fundamentally different architectures.
4. **Limbal transition:** Show fibers curving from radial to tangential within ~1.5 mm of limbus.
5. **In keratoconus:** Show fibers reorienting from 90°/180° to 60°/120° at cone, with slippage gaps between lamellae.

## 3. Pathological States — Visual Standards

### Keratoconus Progression (What to Show)

| Stage | Posterior | Anterior | Epithelium | Plácido |
|-------|-----------|----------|------------|---------|
| **Normal** | Flat, organized | Dense mesh | Uniform ~50 µm | Concentric |
| **Subclínico** | Posterior surface displaces anteriorly (focal) | Normal appearing | Begins thinning over apex | Near-normal |
| **Moderado** | Clear anterior displacement of posterior surface, focal thinning | Redistribution visible, some loss of weave | Donut pattern: thin at apex, thick around | Oval/compressed |
| **Avançado** | Significant anterior displacement, scarring begins | Weave disrupted, fiber reorientation 60°/120° | Extreme compensation | Duck/Snowman |

### CRITICAL PATHOLOGY RULES

1. **Posterior surface in ectasia BULGES ANTERIORLY (toward the anterior stroma)** — it does NOT protrude posteriorly/outward. This is the most common illustration error.
2. **Collagen is REDISTRIBUTED, not destroyed.** Show fibers displaced and reoriented, not absent. Total collagen mass decreases only ~5%.
3. **Epithelial donut pattern:** Thinning at apex + annular thickening = masking of true stromal surface.
4. **Fiber reorientation at cone:** Change from orthogonal (90°/180°) to non-orthogonal (60°/120°).
5. **Interlamellar gaps:** Show increased spacing between lamellae at the cone (proteoglycan loss = "glue" dissolution).

### Post-ICRS State (What to Show)

| Element | Representation |
|---------|---------------|
| ICRS cross-section | Triangular (Ferrara) or hexagonal shape at 70-75% depth |
| Tenting effect | Anterior surface elevated directly above ICRS |
| Arc-shortening | Fibers over ICRS follow a longer path → tension redistribution |
| Von Mises stress | Concentration zones at ICRS edges and limbus |
| Coupling effect | Steepening 90° away from ICRS (tension redistribution, NOT IOP) |

## 4. Visual Style Standards for the Atlas

### Mandatory Elements in Every Corneal Illustration

1. **Scale bar** or labeled thickness measurements
2. **Layer labels** with correct proportions
3. **Fiber direction indicators** (arrows or line patterns)
4. **Portuguese labels** (primary) with optional English in parentheses
5. **Clean medical atlas style** — Netter/Thieme reference aesthetic
6. **No decorative elements** — scientific accuracy over artistic flair
7. **Consistent color palette** across all chapters (see table in Section 1)

### Image Checklist Before Publishing

- [ ] Anterior/posterior thickness ratio is ~1:3 (not 1:1)
- [ ] Anterior fibers shown as interwoven mesh, not parallel
- [ ] Posterior fibers shown as parallel/ordered, not interwoven
- [ ] Bowman's and Descemet's are thin lines, not thick bands
- [ ] Epithelium is thin layer, not exaggerated
- [ ] In KC: posterior displaces anteriorly, not outward
- [ ] In KC: fibers reoriented and slipped, not "broken" or "absent"
- [ ] ICRS placed at 70-75% depth (within posterior stroma)
- [ ] Limbal fibers shown as tangential/circumferential
- [ ] Scale or measurements included
